Documentação

Display

A primeira propriedade que precisa ser declarada para que o Flexbox funcione!

Algumas propriedades como flex-direction, justify-content e aling-content já vem com os valores padrão.

Propriedade Valor
Display Flex
Justify Content flex-start, flex-end, center, space-between, space-around, space-evenly, initial, inherit
Flex Direction row, row-reverse, column, column-reverse, initial, inherit
Aling Itens normal, stretch, center, flex-start, flex-end, start, end, baseline, initial, inherit
Flex Wrap nowrap, wrap, wrap-reverse, initial, inherit
Flex Flow flex-direction, flex-wrap, initial, inherit
Align Content stretch, center, flex-start, flex-end, space-between, space-around, space-evenly, initial, inherit

Propriedades
Display

display

Esta propriedade define um container flex, inline ou block dependendo do valor recebido. Isto habilita o flex para todos os filhos de um elemento pai

                        .container {
                            display: flex;
                        }

Justify-content

flex-direction

Esta propriedade define o alinhamento no eixo horizontal. Isto ajuda a distribuir espaço livre de sobra quando os itens numa linha são inflexíveis, ou são flexíveis mas alcançaram seu tamanho máximo. Também excerce um controle sobre o alinhamento quando os itens transbordam a linha.

                        .container {
                            justify-content: flex-start | flex-end | center | space-between | space-around | space-evenly;
                        }

Flex-wrap

flex-wrap

Por padrão, itens flex vão tentar se encaixar em uma linha. Você pode alterar isso para permitir que os itens sejam "envolvidos" por esta propriedade conforme a necessidade.

                        .container {
                            flex-wrap: nowrap | wrap | wrap-reverse;
                        }
